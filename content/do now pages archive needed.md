date: 2026-06-11

读 David C. Buchan 的 [now page](https://quantumgardener.info/now) 有感。

自2023年初开始记录 [[now|now page]] 以来，按照平均每周更新一次的频率，至今已有 ~~一坤年~~ 两年半，大约120周。而我上一个数字花园是 Node.js  TiddlyWiki 版本，每次更新 now page 都会覆盖上一个版本，只有少量定期上传至 GitHub Page 的备份和手动导出备份。

David C. Buchan 的 now page 中列出他每一次修改 now page 的快照，标题格式为是 [Now, DD MM YYYY]，按照时间倒序排列。这给了我一种新的思路，是追溯过去某一个时刻切片的优秀样本。

闲时我将尝试收集留存下来的记录，整理到 now pages archive 页面里。
