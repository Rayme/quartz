I use Obsidian and Anytype (not anymore) mostly. Obsidian for all my work and study, Anytype for my daily journals, and I use eDiary for my daily diaries. This setup is almost perfect for me.

I organize daily journals every once a month or so, sometimes longer. I organize them to a file in Obsidian and display by week. I delete and rewrite some notes of them. I don't know about others but it looks good for me for now.

I don't like those note taking software that lock people in, like Evernote or Notion. But I do use Notion. I use it only as a bookkeeping (not anymore), but I can discard it at anytime. So that I don't feel bad if I lost it or give away my privacy.

TiddlyWiki is great software too. But I don't use it much these days. I just don't find a proper use case for me. I only use it as a daily work log, but not primary.
